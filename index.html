Multi-Model Routing

Route to OpenAI, Anthropic, Azure, or local models per-agent. Automatic fallback when a provider fails. No vendor lock-in.

🛡️

Native Dashboard

Hlidskjálf — a native SwiftUI app for macOS and iOS. Real-time agent monitoring, event feeds, and command dispatch. No Electron bloat.

Discord Command & Control

Channel-based agent routing, approval workflows, planner-to-dev handoffs. Your Discord server becomes mission control.

"Hey Allsyn" — speech-to-text, conversational AI, and text-to-speech. On-device processing for privacy. Discord voice channel support.

One command converts your openclaw.json to Allsyn format. Agents, routes, channels, tokens — all migrated automatically.
